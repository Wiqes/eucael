# Notification Sound

This file should contain a notification sound (notification.mp3) for the chat system.

For development purposes, you can:

1. Add a short notification sound file named `notification.mp3` to this directory
2. Use a royalty-free notification sound
3. The audio file should be short (1-2 seconds) and pleasant

Common notification sound formats supported:

- .mp3
- .wav
- .ogg

The sound will be played when:

- A new message arrives when the app is not in focus
- The user receives a notification while the browser tab is not active
